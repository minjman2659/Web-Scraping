# Web-Scraping
Web-scraping by Nodejs (Base _ Naver news)

[*참고](https://www.scrapingbee.com/blog/web-scraping-javascript/)

<br />

## 💻 사용한 스택

Express, Axios, Cheerio, Puppeteer

<br />

## 📌 실행 방법

Node 16 혹은 그 이상의 버전을 필요로 합니다.

```javascript
 - yarn // install dependencies
 - yarn dev // run server
```

<br />
